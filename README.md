## Build
* Clone this repository:
```bash
git clone git@abbradar.moe:promestein
```

* Clone it's submodules:
```bash
git submodule init
git submodule update
```

* In the `data.xml` directory build and install the project:
```bash
cd ./checkouts/data.xml
lein deps
lein install
```
